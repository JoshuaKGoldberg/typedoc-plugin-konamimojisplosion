## Usage

Add this plugin in your `typedoc.json`:

```json
{
	"plugin": ["typedoc-plugin-konamimojisplosion"]
}
```

Then, pressing the [Konami Code](https://en.wikipedia.org/wiki/Konami_Code) on your TypeDoc site will trigger fireworks blasts of emoji on the page.
